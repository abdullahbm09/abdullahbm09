## Highlighted Work

### Red Team Assessment — Cloud-Robotics Environment (Dell-Funded, 2024–2025)
Full offensive security engagement against a live smart manufacturing simulation.

- **Chain 1:** ROS2 permission misconfigurations → unauthorised pub/sub access → adversarial manipulation of deployed AI defect detection model
- **Chain 2:** CVE-2023-38408 (SSH agent forwarding) → lateral movement + RCE → Python sudo privilege escalation → root
- Modelled using MITRE ATT&CK TTPs · Quantitative security metrics delivered to Dell Technologies

→ [`cloud-robotics-redteam`](https://github.com/abdullahbm09/cloud-robotics-redteam)

### DD-IDS — Data-Driven IDS for IIoT Smart Factories (PhD Research)
PCA with Hotelling T² and SPE statistics detecting false data injection attacks on industrial control systems. Validated against adversarial attacks on the Tennessee Eastman process. Detection and automated mitigation embedded into blockchain smart contracts.

→ [`dd-ids-iot`](https://github.com/abdullahbm09/dd-ids-iot)

### Website Backdoor — Incident Response & Remediation (2025)
Investigated and fully remediated a persistent backdoor: database-level SEO spam injection across 1,448 malicious posts, CSS-obfuscated hidden links, rogue admin account creation. Forensically scanned 19,855 files. Full incident report and hardening plan produced.
